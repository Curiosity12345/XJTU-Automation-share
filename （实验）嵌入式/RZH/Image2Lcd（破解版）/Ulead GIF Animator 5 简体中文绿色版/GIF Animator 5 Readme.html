Introduction

Ulead GIF Animator is a powerful GIF animation tool for creating high impact animation for Web pages, presentations, and multimedia titles. From composition, editing, and applying special effects, you can do them all in GIF Animator. GIF Animator also gives you total control over optimization, allowing you to make animations compact and web-ready while retaining overall image quality. Moreover, when it is time to produce the final animation file, the program lets you output your animation in a variety of file formats. File formats supported include animated GIF, Windows AVI, QuickTime, Autodesk animation, image sequences, and even Flash animation.

System Requirements

System requirements include the following:

  • Microsoft Windows® 95, Windows® 98, Windows® NT 4.0 SP5 (or above), Windows® 2000, or Windows® ME
  • Intel® Pentium® Processor or compatible
  • 64 MB RAM
  • 20 MB available hard disk space for program installation
  • True Color or Hi-Color display adapter and monitor, with 800x600 (recommended) or higher display mode
  • Windows-compatible pointing device

Installing/Uninstalling Ulead GIF Animator 5

  • After you have downloaded the zipped Ulead GIF Animator 5 ESD version, double-click the executable program and follow the installation instructions.
  • You can uninstall GIF Animator 5 via Add/Remove Programs in the Control Panel.

Installing Internet Explorer 5.5

The Preview function in Ulead GIF Animator 5 will only be available when Microsoft® Internet Explorer® is installed in your system. Ulead strongly recommends that you install Internet Explorer® 5.5 or above to fully utilize GIF Animator’s features. Internet Explorer can be downloaded from Microsoft Website (http://www.microsoft.com).

Installing DirectX Media 6.0

If you would like to open or export AVI files in Ulead GIF Animator 5, please install Microsoft DirectX Media 6.0 or above. You can download DirectX Media 6.0 from Microsoft Website (http://www.microsoft.com).

Note: If GIF Animator 5 fails to launch after installing DirectX Media, please re-install GIF Animator 5 and use the Repair function then run GIF Animator 5 again.

Installing QuickTime 4.1

If you would like to open or export Apple Quicktime (*MOV and *.QT) files in Ulead GIF Animator 5, please install QuickTime 4.1 or above. You may download the QuickTime software from Apple Website(http://www.apple.com/quicktime/).
